# Inbox

Staging area for non-destructive AI memory work before promotion into canonical vault files.

- `proposals/` — summarize proposals generated by `ai-vault summarize`.
- `session-summaries/` — short append-only session summaries paired with proposals.

Rules:

- Do not commit raw secrets.
- Do not keep raw chat logs long-term.
- Mark private project memory as private by default.
- Promote useful material into `projects/`, `skills/`, `patterns/`, or `global/` only after review and validation.
